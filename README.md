# Automated MIS Dashboard

## Overview

This project presents an interactive MIS dashboard created using Microsoft Excel to monitor key business metrics such as sales performance and targets.

## Features

* KPI indicators (Total Sales, Target, Achievement %)
* Department-wise performance tracking
* Interactive slicers for filtering data
* Visual charts for better insights

## Tools Used

* Microsoft Excel (Pivot Tables, Charts, Slicers)

## Outcome

The dashboard improves visibility of performance metrics and enables quick decision-making through interactive data visualization.
